# Project-Book-My-Ticket

Welcome to the **Project-Book-My-Ticket** repository! This is a full-stack event booking and management application built using the MERN stack.

## 📁 Repository Structure
- `Book-My-Ticket-Backend/`: Express.js server, MongoDB models, Stripe integration, and PDF ticketing logic.
- `Book-My-Ticket-Frontent/`: React 19 frontend with Tailwind CSS 4, Recharts dashboard, and real-time booking UI.

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MongoDB URI
- Stripe API Keys
- Cloudinary Credentials

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vivekp-codes/Project-Book-My-Ticket.git
   ```

2. **Setup Backend:**
   ```bash
   cd Book-My-Ticket-Backend
   npm install
   # Create a .env file based on provided keys
   npm start
   ```

3. **Setup Frontend:**
   ```bash
   cd ../Book-My-Ticket-Frontent
   npm install
   npm run dev
   ```

## ✨ Key Features
- **Event Management**: Create and manage events seamlessly.
- **Secure Payments**: Integrated Stripe checkout.
- **Digital Ticketing**: PDF tickets with QR codes for entry.
- **Admin Dashboard**: Real-time analytics and booking insights.
